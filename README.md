# Inventory-Tracking-System
 📦 Inventory Management System (Version 2)

A console-based **Inventory Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles. The system manages products, stock items, warehouses, and sales orders while storing data permanently using CSV files.

🚀 Features

 Product Management
- Add new products
- View all products
- Search products by ID
- Update product details
- Remove products

Stock Item Management
- Add stock items
- Track available inventory
- Update stock quantities
- Remove stock items
- Prevent duplicate product entries

Sales Order Management
- Create sales orders
- Validate stock availability before processing orders
- Automatically update stock quantity after a successful sale

 Data Storage
- Read data from CSV files when the program starts
- Save updated data to CSV files before exiting
- Persistent storage without using a database

 Additional Features
- Input validation
- Menu-driven console interface
- Colorful console output for a better user experience


🛠 Technologies Used

💡 OOP Concepts Implemented

- Classes and Objects
- Encapsulation
- Composition
- Arrays of Objects
- Modular Programming
- File Handling
- Input Validation

📁 CSV Files

The project stores data using CSV files:

- Products.csv
- StockItem.csv
- SalesOrder.csv

This allows data to persist between program executions.
